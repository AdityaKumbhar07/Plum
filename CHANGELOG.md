## [1.0.0] - 2026-06-21

### Added
- **Core Database & Architecture**: Fully offline SQLite database using Drift.
- **Customer Management**: Ability to add customers and view their history.
- **Quick Bill (बिल)**: Cart-style billing system with instant PDF generation and WhatsApp sharing.
- **Quotation System**: Create material estimates and send via formatted WhatsApp text.
- **Payment Tracker**: Dashboard to view unpaid balances with one-tap WhatsApp payment reminders.
- **Core Cutting Calculator**: Interactive rate card calculator with clipboard copying.
- **Work Showcase**: Built-in photo gallery for plumbing and core cutting work photos.
- **Bilingual Support**: Instant toggle between English and Marathi languages.
- **Settings**: Manage owner profile details and default application preferences.
